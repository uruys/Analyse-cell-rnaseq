Le but de ce projet était d’analyser des données single-cell RNA-seq afin d’identifier differents groupes de cellules et d’observer leurs profils d’expression génique à l’aide d’outils bioinformatiques.


D'après le graphique UMAP on peut observer plusieurs clusters différents cela indique la presence de differentes populations cellulaires autrement dit les clusters.Puis la heatmap montre que certains gènes sont plus exprimés dans certains clusters, ce qui permet de distinguer les differents groupes de cellules. Ces résultats montrent une hétérogénéité cellulaire, ce qui signifie que les cellules ont des profils d’expression génique differents.

Ce projet m’a permis d’utiliser des outils bioinformatiques pour analyser des données single-cell RNA-seq et d’identifier differents groupes de cellules et leurs gènes marqueurs.


- Python
- Scanpy
- Matplotlib
- Jupyter Notebook (Anaconda)
